| ID | 名称 | 说明 |

| 1 | 战斗开始 | 战斗开始时触发 |
| 2 | 回合开始 | 每回合开始时触发 |
| 3 | 回合结束 | 每回合结束时触发 |
| 4 | 攻击 | 跟随主动技能，伤害结算前触发 |
| 7 | 命中 | 跟随主动技能，命中后触发 |
| 8 | 被命中 | 受击时触发 |
