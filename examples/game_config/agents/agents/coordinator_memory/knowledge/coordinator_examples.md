# 主策划案例库

> 做过的案例，供 AI 类比新需求时参考。

## 案例 1：UR 红发装备套装（2026-03-10）

**需求类型**：UR装备
**涉及模块**：12+1个（装备/属性/强化/套装/道具投放等）
**涉及角色**：战斗策划（套装buff）+ 数值策划（强化数值）+ 执行策划（全表写入）
**涉及表**：13张

**拆分方式**：
1. 主策划先列出 12+1 模块清单，用户确认
2. 战斗策划设计套装效果 buff（2件套/4件套）+ 增强 buff（4级）
3. 数值策划确认 BuffActive 参数值
4. 执行策划按 SOP 逐表写入

**教训**：首次漏了模块确认环节，直接跳到表级别 → 铁规"功能模块先确认"

## 案例 2：破绽被动技能（2026-03-11）

**需求类型**：被动技能
**涉及角色**：战斗策划 + 执行策划
**涉及表**：3张（_Buff / BuffActive / FightBuff）

**拆分方式**：
1. 战斗策划按四要素拆解：2句话 → 2条链路
2. 链路1：命中→施加破绽（爆伤-50%，2回合）+ 关联计数标记
3. 链路2：被暴击→叠加标记→每2层触发削HP
4. 执行策划填表：_Buff 4行 + BuffActive 4行 + FightBuff 2行

## 案例 3：新阵法（2026-03-11）

**需求类型**：新阵法
**涉及角色**：战斗策划 + 执行策划
**涉及表**：8张（Formation → FormationPos → FormationLev → Item → FightBuff → _Buff → BuffActive → _BuffCondition）

**拆分方式**：
1. 战斗策划设计阵法 buff 效果
2. 执行策划按 SOP 逐表写入（6步）
