# 装备系统

> 来源：怀旧服数值总表.xlsx

## 1. 装备数值设计原则 [Sheet:怀旧服装备]
1. 前期装备数值提升明显
2. 高等级装备有更有吸引力的数值
3. 高等级普通装备 ≥ 低一等级英雄装备基础数值

## 2. 装备精炼 [Sheet:装备精炼/装备精炼(2)]
| 精炼等级 | 攻击 | 防御 | 生命 |
|---|---|---|---|
| 1 | +30 | +12 | +240 |
- 1攻击花费彩钻 = 13.33
- 精炼(2)改动：去掉精炼石种类改为货币概念，提升成功率降低消耗

## 3. 装备词条 [Sheet:装备词条/词条生成]
- 分等级阶段(10/20/40/50/60级)
- 词条按职业定位差异化：
  | 职业 | 命中 | 闪避 | 暴击 | 韧性 | 格挡 | 破击 | 属伤 | 属抗 | 速度 |
  |---|---|---|---|---|---|---|---|---|---|
  | 物理坦克 | 50 | 50 | - | 100 | 100 | - | - | - | 50 |
  | 元素坦克 | 50 | 50 | - | 50 | 50 | - | - | 100 | 50 |
  | 元素战士 | 70 | - | 50 | 50 | - | 30 | 100 | - | 50 |

## 4. 套装效果 [Sheet:套装效果/套装效果(2)/套装效果库]
- 2件套：属性百分比加成(攻击8%/生命12%/韧性10%等)
- 4件套：战斗特效(护盾/减伤/属伤提升等)
- buffId映射：如22014(卫兵4件套)/22010(威风4件套)

## 5. 装备晋升 [Sheet:绝境挑战]
- 60级装备可晋升，消耗晋升石
- 晋升到70/80级可触发共鸣
