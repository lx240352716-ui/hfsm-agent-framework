# 🌟 主策划模块经验模板库 (Templates)

这个目录专门用于存放**主策划总结的结构级经验**。
当完成一次复杂的复合型需求（如：节日活动、新系统培养等）后，主策划必须将这套**涉及的几十张表的清单和组合规律**，精炼成一个模块化组合套餐。

## 储存格式

请新建 `.md` 文件，命名为 `[大功能名称]_template.md`，内容必须包含：

1. **触发条件**：什么时候用这套模板？
2. **需要分配的模块矩阵**：例如
   - 基础道具模块 -> 需要改 `表A`, `表B`
   - UI红点模块 -> 需要改 `表C`
   - 数值掉落模块 -> 需要改 `表D`
3. **主策划叮嘱**：这个模块在分发给下层角色时，有哪些特殊交代？（如：表A记得和表C的某字段保持一致）
